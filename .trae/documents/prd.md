## 1. 产品概述

展厅数字孪生管理系统是一个基于 3D 可视化的智能监控平台，通过在网页中渲染展厅的三维模型，并以热力球体的形式实时展示各展位的客流量数据，帮助管理人员直观掌握展厅运营状况，辅助优化展位布局和人流管控。

- 目标用户：展厅运营管理人员、活动策划人员
- 核心价值：将抽象的客流数据转化为直观的 3D 热力可视化，提升展厅管理效率

## 2. 核心功能

### 2.1 功能模块

1. **3D 展厅看板页**：3D 场馆模型、展位热力球体、客流量图例面板、实时数据刷新

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 3D 展厅看板页 | 3D 场馆模型 | 用方块渲染展厅地面与墙体，展示空间布局 |
| 3D 展厅看板页 | 展位热力球体 | 在各展位坐标处渲染球体，颜色与大小根据客流量变化（绿→黄→红） |
| 3D 展厅看板页 | 交互控制 | 支持鼠标拖拽旋转、滚轮缩放、点击展位查看详情 |
| 3D 展厅看板页 | 图例面板 | 左侧面板展示客流量颜色图例与统计数据 |
| 3D 展厅看板页 | 实时刷新 | 每 30 秒自动从后端拉取最新客流数据 |

## 3. 核心流程

用户进入系统后，自动加载 3D 展厅场景。前端从后端 API 获取展位坐标与客流量数据，在 3D 空间对应位置渲染热力球体。用户可通过鼠标旋转/缩放查看展厅全貌，点击展位球体可查看详细客流信息。系统每 30 秒自动刷新数据。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色科技风（#0a0e1a 深空蓝黑底色 + #00e5ff 赛博青色强调色 + #ff3d71 热力红色警示色）
- **按钮样式**：圆角半透明毛玻璃风格，带微光边框
- **字体**：Orbitron（标题/数据展示）+ Noto Sans SC（正文）
- **布局风格**：全屏 3D 画布为主，左侧悬浮信息面板
- **图标风格**：线性极简（Lucide Icons）

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 3D 展厅看板页 | 3D 画布 | 全屏 Three.js 渲染，深色环境光，柔和点光源 |
| 3D 展厅看板页 | 左侧面板 | 毛玻璃背景，标题"展厅监控"，客流量图例（绿/黄/红渐变条），总客流统计 |
| 3D 展厅看板页 | 热力球体 | 半透明发光球体，绿→黄→红颜色映射，球体大小随客流值缩放，hover 高亮 |
| 3D 展厅看板页 | 展位标签 | 球体上方浮动文字标签，显示展位名称与客流数值 |
| 3D 展厅看板页 | 顶部栏 | 系统标题 + 当前时间 + 刷新状态指示器 |

### 4.3 响应式设计

- 桌面端优先，全屏 3D 体验
- 平板端：面板收窄为可折叠侧栏
- 移动端：面板底部抽屉式弹出，3D 画布触控手势操作

### 4.4 3D 场景指导

- **环境与氛围**：深色科技感环境，暗蓝色雾效营造纵深感
- **灯光设置**：AmbientLight 低强度环境光 + DirectionalLight 主光 + PointLight 展位局部补光
- **摄像机**：PerspectiveCamera，初始俯视 45°，OrbitControls 支持旋转/缩放/平移
- **构图与焦点**：展厅模型居中，热力球体为视觉焦点
- **交互与动画**：热力球体呼吸脉冲动画，hover 放大高亮，数据更新时平滑过渡
- **后处理效果**：Bloom 辉光效果增强热力球体发光感
- **性能预算**：60fps 流畅渲染，展位数量 ≤ 50

## 5. 后端接口需求

| 接口 | 方法 | 描述 |
|------|------|------|
| /api/booths | GET | 返回所有展位信息（ID、名称、坐标、尺寸） |
| /api/booths/{id}/traffic | GET | 返回指定展位客流量数据 |
| /api/traffic | GET | 返回所有展位当前客流量汇总 |
